# Prototypes

Design prototypes and concept explorations for IRS Taxpayer Services & Online Accounts.
Self-contained HTML — open any file directly in a browser. Styled to the IRS
Online Design Guidelines (ODG) / U.S. Web Design System. All data shown is mock.

## Published (GitHub Pages)

Published via GitHub Pages so prototypes are easy to share and update — push to
`main` and the site redeploys. The root `index.html` is the prototype index.

## Contents

### `omnibar-in-app/` — Online Account "omni bar" (in-app concept)
A modern Online Account web-app concept with a unified action/search "omni bar."

- `index.html` — walkthrough landing page (screenshots in `pngs/`)
- `online-account-webapp.html` — the omni bar prototype
- `online-account-webapp-banner.html` — banner variant

### `bta-94x-pin/` — Business Tax Account: 94x Online Signature PIN
A self-service wizard for a responsible official to request the 10-digit 94x
Online Signature PIN (Forms 940, 941, 943, 944, 945) inside the Business Tax
Account, then e-file straight to MeF.

- `index.html` — the PIN application prototype
